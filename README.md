# JATS

JATS (Java Assistant Tools) using Spring Shell 2 to provide many various command-line utilities to rapidly develop Java
Web Application

## Requirement

* JDK 8
* Maven 2.3.2
* Spring Shell 2.0.1

## Build

```
mvnw clean package
```

## Run

```
java -jar (path_to_jats_project)\target\core-2.0.1.jar
```
